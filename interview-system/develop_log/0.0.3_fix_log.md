# 版本 0.0.3 Bug 修复记录

## 修复时间
2026-02-16

## 修复内容

### 1. LLMConfig 序列化异常
**问题**: `java.io.NotSerializableException: com.interview.model.LLMConfig`

**原因**: LLMConfig 类没有实现 Serializable 接口

**修复**: 
```java
public class LLMConfig implements Serializable {
    private static final long serialVersionUID = 1L;
    // ...
}
```

### 2. ReportView 空指针异常
**问题**: `Cannot invoke "AuthService.hasPermission(...)" because "this.authService" is null`

**原因**: ReportView 构造函数传入 null 给 InterviewRecordService

**修复**:
- 修改 ReportView 构造函数接收 AuthService 参数
- 修改 MainView 中两处创建 ReportView 的地方传递 authService

```java
// ReportView.java
public ReportView(AuthService authService) {
    this.evaluationService = new EvaluationService(authService);
    this.recordService = new InterviewRecordService(authService);
    // ...
}

// MainView.java
addTab("评测报告", new ReportView(authService));
addTab("我的评测报告", new ReportView(authService));
```
